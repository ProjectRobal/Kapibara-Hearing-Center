Model maszynowy który analizuję próbki dźwięku i decyduję czy dźwięk jest:

-straszny
-przyjemny
-wnerwiający (wywołujący agresję) - powodujący gniew
-neutralny
-niepokojący (naprzykład szum)

- strach[^1]
- smutek
- gniew
- przyjemność
- zaniepokojenie
